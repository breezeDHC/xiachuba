# xiachuba

下厨吧是一个综合发布菜谱、发布动态等一体的美食网站。web项目。地址：39.107.91.81:8080。

### 环境

mysql5.6、jdk1.8.0、maven、idea、tomcat8.5

### 项目介绍

这是一个比较完整的web项目，使用了三层架构，前后端分离技术，并实现了上线。主要的功能包括登录、注册、后台管理、搜索、创建菜谱、发表动态、个人中心、评论、留言、点赞、收藏、关注、各种信息展示等功能。

### 项目技术

validate、Layer框架、HTML、CSS、JavaScript、jQuery、ajax、json、文件上传uploadFile、短信验证接口、滑块验证插件、md5加密、jsbc、c3p0、maven、git、mysql、分页、javaEE、session。

### 项目角色

这是一个团队项目， 我在项目中独立完成关注和菜谱创建功能，这也是项目中的重点和难点功能。参与完成项目业务分析、数据库设计、项目合并、项目上线、性能测试，帮助部分队友解决问题。
